Classification of Heart Failure Replication Project


I found a dataset on UCI Machine Learning that covers heart failure. This dataset is discussed in the paper: "Machine learning can predict survival of patients with heart failure from serum creatinine and ejection fraction alone," so I decided to replicate the analysis in this paper in my project. I will predict the survival of patients with heart failure using a variety of classifiers, perform data analysis using hypothesis testing and perform feature ranking.



The results of the analysis can be found in the following workbook: Analysis Workbook
